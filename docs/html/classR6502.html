◆ modeAbsolute()

void R6502::modeAbsolute ( )
private

Absolute addressing mode.

Second byte has low order byte of effective address, third has higher order byte (reminder again: 6502 is little endian!!!) That means ABS instructions can acess all 64KB of address space. 2^8 = 0x100 2 byte addressing... 0x100 * 0x100 = 0x10000 (64KB)
